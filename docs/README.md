# Kiroku Documentation

This folder is the working developer reference for the current app structure.

Files:

- `pages.md`: route map, frontend page responsibilities, and shared UI patterns.
- `services.md`: backend services, chat integration, Redis, Ollama, and proxying.
- `developer-guide.md`: local setup, startup order, and practical workflow notes.

Keep these files updated when routes, service ports, startup behavior, or cross-service contracts change.
